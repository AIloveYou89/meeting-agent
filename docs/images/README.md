# Ảnh hướng dẫn — danh sách cần chụp

Thư mục này chứa ảnh minh họa cho `docs/01-tao-google-cloud.md`.
Anh chụp màn hình theo danh sách dưới, **đặt đúng tên file**, bỏ vào thư mục này là ảnh tự hiện trong hướng dẫn.

## ⚠️ Trước khi chụp — CHE các thông tin sau (repo này công khai!)
- **Email / avatar** góc trên bên phải (làm mờ hoặc bôi đen).
- **Project ID / Project number** (dạng `project-xxxx` hoặc dãy số) — bôi đen.
- Bất kỳ số thẻ, số tiền, thông tin cá nhân nào.
- Mẹo: dùng công cụ chụp có sẵn (Mac: Shift+Cmd+4) rồi tô đen bằng Preview/Paint.

## Danh sách 5 ảnh cần chụp

| Tên file (đặt đúng) | Chụp màn hình nào | Ứng với mục |
|---------------------|-------------------|-------------|
| `01-new-project.png` | Màn hình **New Project** — ô điền tên project | 1.2 |
| `02-enable-api.png` | Trang **Speech-to-Text API** có nút **Enable** (hoặc sau khi đã Enabled) | 1.3 |
| `03-billing.png` | Trang **Billing** — chỗ "Link a billing account" (che số thẻ) | 1.4 |
| `04-create-service-account.png` | Màn **Create Service Account** — điền tên + chọn Role `Cloud Speech Client` | 1.5 |
| `05-tao-key-json.png` | Tab **Keys** → menu **Add Key → Create new key → JSON** | 1.5 |

## Định dạng
- Nên dùng **PNG** (nét chữ rõ). Kích thước vừa phải (< 1MB/ảnh là đẹp).
- Nếu muốn khoanh vùng "bấm vào đây", vẽ 1 khung đỏ đơn giản cho dễ nhìn.

> Chưa có ảnh cũng không sao — hướng dẫn bằng chữ đã đủ chi tiết để làm theo.
